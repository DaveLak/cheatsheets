### Typically C:\Windows

```
%SYSTEMROOT%
```

### DNS entries
```
%SYSTEMROOT%\System32\drivers\etc\hosts
```

```

%SYSTEMROOT%\repair\SAM
```

```
%SYSTEMROOT%\System32\config\RegBack\SAM
```

```
%WINDIR%\system32\config\AppEvent.Evt
```

### Security Log
```
%WINDIR%\system32\config\SecEvent.Evt
```
