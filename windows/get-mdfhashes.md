### Get hash from mdf mssql database file
```
Import-Module .\OrcaMDF.RawCore.dll
Import-Module .\OrcaMDF.Framework.dll
Import-Module .\Get-MDFHashes.ps1
Get-MDFHashes -mdf "<fullPathToFile>.mdf"
```
