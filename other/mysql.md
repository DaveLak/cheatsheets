### Show all databases
```
show databases;
```

### Switch context
```
use <database>;
```

### Show tables
```
show tables;
```

### Print content of table
```
select * from <table>;
```

### Execute command
```
select do_system('<command>');
```

### Read file
```
select load_file('<file>');
```

### Wite to file
```
select "<content>" into outfile '<file>';
```
